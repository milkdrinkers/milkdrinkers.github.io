# Installation

## Adding Dependency

In order to use ColorParser you will need to download it using a BuildTool like Maven or Gradle.

<Tabs items={['Gradle Kotlin DSL', 'Gradle Groovy', 'Maven']}>
    <Tabs.Tab>
        <Steps>
        ### Add Repository

        ```kotlin
        repositories {
            maven("https://jitpack.io")
        }
        ```

        ### Add Dependency

        ```kotlin
        dependencies {
            implementation("com.github.milkdrinkers:colorparser:VERSION")
        }
        ```

        ### Shade Dependency

        ```kotlin
        plugins {
            id("com.github.johnrengelman.shadow") version "8.1.1"
        }

        tasks {
            shadowJar {
                relocate("com.github.milkdrinkers.colorparser", "yourpackage.yourname.colorparser")
            }
        }
        ```
        </Steps>
    </Tabs.Tab>
    <Tabs.Tab>
        <Steps>
        ### Add Repository

        ```groovy
        repositories {
            maven { url 'https://jitpack.io' }
        }
        ```

        ### Add Dependency

        ```groovy
        dependencies {
            implementation 'com.github.milkdrinkers:colorparser:VERSION'
        }
        ```

        ### Shade Dependency

        ```groovy
        plugins {
            id 'com.github.johnrengelman.shadow' version '8.1.1'
        }

        tasks {
            shadowJar {
                relocate 'com.github.milkdrinkers.colorparser', 'yourpackage.yourname.colorparser'
            }
        }
        ```
        </Steps>
    </Tabs.Tab>
    <Tabs.Tab>
        <Steps>
        ### Add Repository

        ```xml
        <repositories>
            <repository>
                <id>jitpack.io</id>
                <url>https://jitpack.io</url>
            </repository>
        </repositories>
        ```

        ### Add Dependency

        ```xml
        <dependency>
            <groupId>com.github.milkdrinkers</groupId>
            <artifactId>colorparser</artifactId>
            <version>VERSION</version>
        </dependency>
        ```

        ### Shade Dependency

        ```xml
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-shade-plugin</artifactId>
            <version>3.5.0</version>
            <executions>
                <execution>
                    <phase>package</phase>
                    <goals>
                        <goal>shade</goal>
                    </goals>
                </execution>
            </executions>
            <configuration>
                <createDependencyReducedPom>false</createDependencyReducedPom>
                <relocations>
                    <relocation>
                        <pattern>com.github.milkdrinkers.colorparser</pattern>
                        <shadedPattern>yourpackage.yourname.colorparser</shadedPattern>
                    </relocation>
                </relocations>
            </configuration>
        </plugin>
        ```
        </Steps>
    </Tabs.Tab>
</Tabs>

## Platforms Other Than PaperMC

If using Spigot (*or any other platform that doesn't natively ship Adventure*), then you will need to add Adventure as a dependency in your project.

Check supported platforms [here](https://docs.advntr.dev/platform/native.html). And versions [here](https://github.com/KyoriPowered/adventure/releases).

<Tabs items={['Gradle Kotlin', 'Gradle Groovy', 'Maven']}>
    <Tabs.Tab>
        <Steps>
        ### Add Dependency

        ```kotlin
        dependencies {
            implementation("net.kyori:adventure-api:VERSION")
            implementation("net.kyori:adventure-text-minimessage:VERSION")
        }
        ```
        </Steps>
    </Tabs.Tab>
