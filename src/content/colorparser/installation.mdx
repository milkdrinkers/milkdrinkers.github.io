# Installation

## Adding Dependency

In order to use ColorParser you will need to download it using a BuildTool like Maven or Gradle.

The currently supported platforms are:
- `io.github.milkdrinkers:colorparser-bukkit`
- `io.github.milkdrinkers:colorparser-paper`
- `io.github.milkdrinkers:colorparser-velocity`
- `io.github.milkdrinkers:colorparser-sponge8`
- `io.github.milkdrinkers:colorparser-sponge9`
- `io.github.milkdrinkers:colorparser-sponge10`
- `io.github.milkdrinkers:colorparser-sponge11`
- `io.github.milkdrinkers:colorparser-sponge12`

For a full list of platforms and versions [check out Maven Central here](https://central.sonatype.com/search?q=colorparser&namespace=io.github.milkdrinkers).

<Tabs items={['Gradle Kotlin DSL', 'Gradle Groovy', 'Maven']}>
    <Tabs.Tab>
        <Steps>
        ### Add Repository

        ```kotlin filename="build.gradle.kts"
        repositories {
            mavenCentral()
        }
        ```

        ### Add Dependency

        ```kotlin filename="build.gradle.kts"
        dependencies {
            implementation("io.github.milkdrinkers:colorparser-PLATFORM:VERSION")
        }
        ```

        ### Shade Dependency

        ```kotlin filename="build.gradle.kts"
        plugins {
            id("com.gradleup.shadow") version "8.3.6"
        }

        tasks {
            shadowJar {
                relocate("io.github.milkdrinkers.colorparser", "yourpackage.yourname.colorparser")
            }
        }
        ```
        </Steps>
    </Tabs.Tab>
    <Tabs.Tab>
        <Steps>

        ```groovy filename="build.gradle"
            mavenCentral()
        }
        ```

        ### Add Dependency

        ```groovy filename="build.gradle"

        }
        ```

        ### Shade Dependency

        ```groovy filename="build.gradle"

            id 'com.gradleup.shadow' version '8.3.6'
        ```

        </Steps>
    </Tabs.Tab>
    <Tabs.Tab>
        <Steps>
        ### Add Dependency

        ```xml filename="pom.xml"
        <dependency>
            <groupId>io.github.milkdrinkers</groupId>
            <artifactId>colorparser-PLATFORM</artifactId>
            <version>VERSION</version>
        </dependency>
        ```

        ### Shade Dependency

        ```xml filename="pom.xml"
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-shade-plugin</artifactId>
            <version>3.5.0</version>
            <executions>
                <execution>
                    <phase>package</phase>
                    <goals>
                        <goal>shade</goal>
                    </goals>
                </execution>
            </executions>
            <configuration>
                <createDependencyReducedPom>false</createDependencyReducedPom>
                <relocations>
                    <relocation>
                        <pattern>io.github.milkdrinkers.colorparser</pattern>
                        <shadedPattern>yourpackage.yourname.colorparser</shadedPattern>
                    </relocation>
                </relocations>
            </configuration>
        </plugin>
        ```
        </Steps>
    </Tabs.Tab>
</Tabs>

## Platforms That Don't Natively Ship Adventure

If using Spigot (*or any other platform that doesn't natively ship Adventure*), then you will need to add Adventure as a dependency in your project.

Check supported platforms [here](https://docs.advntr.dev/platform/native.html). And versions [here](https://github.com/KyoriPowered/adventure/releases).

<Tabs items={['Gradle Kotlin', 'Gradle Groovy', 'Maven']}>
    <Tabs.Tab>
        <Steps>
        ### Add Dependency

        ```kotlin filename="build.gradle.kts"
        dependencies {
            implementation("net.kyori:adventure-api:VERSION")
            implementation("net.kyori:adventure-text-minimessage:VERSION")
        }
        ```
        </Steps>
    </Tabs.Tab>
