# Introduction

Java-Semver is a lightweight, zero-dependency Java library that provides full support for Semantic Versioning 2.0.0. Designed for simplicity and reliability, it enables parsing and comparing semantic versions (major.minor.patch-preRelease+build) effortlessly.
